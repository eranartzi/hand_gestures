This data set is a derivation of the gestures dataset found here: https://github.com/mon95/Sign-Language-and-Static-gesture-recognition-using-sklearn

It is comprised of ~1600 pictures of annotated hand gestures representing the ABC. In the original dataset the pictures are bigger,
including the arm, shoulder, more background etc - the pictures in this fil are already cropped to fit only the gesture area.

The file is a pickle format comprising of a list with the following shape [(pic_date, label)]

Enjoy
